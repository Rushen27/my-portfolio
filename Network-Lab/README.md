# Network Lab – Ubuntu Server, SSH, Apache

This lab was completed as part of my university practical work to learn server setup and network services.

## Tasks Completed

* Installed Ubuntu Server on VMware Workstation
* Configured SSH server for remote login
* Installed and configured Apache2 Web Server
* Tested server connectivity using local network
* Managed Linux users and permissions
* Tested file transfer and server access

## Technologies Used

* Ubuntu Server
* VMware Workstation
* SSH
* Apache2 Web Server
* Linux Command Line

## Commands Used

* sudo apt update
* sudo apt install openssh-server
* sudo apt install apache2
* sudo systemctl start apache2
* sudo systemctl enable apache2

## Outcome

Successfully configured an Ubuntu server with SSH remote access and a running Apache web server. This lab helped me understand basic server administration, networking, and security configuration.
